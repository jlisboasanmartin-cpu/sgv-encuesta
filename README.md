# SGV Encuesta - Estrategia en Salud UAndes 2026

## Cómo actualizar el Form ID de Formspree
1. Abre index.html
2. Busca YOUR_FORM_ID
3. Reemplaza con tu ID real de Formspree
4. Guarda, haz commit y push -> Vercel despliega automático

## Cómo modificar las preguntas
Todas las preguntas están en index.html, dentro de los divs con clase "step".
- step-1: identificación (nombre)
- step-2: hipótesis estratégica
- step-3: premortem + aporte para pregunta final
